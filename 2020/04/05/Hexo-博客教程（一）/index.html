Hexo 博客教程

Hexo 博客教程（一）

基于Windows系统,参考文档：Hexo官网

前期准备：

安装Git； //官网下载
安装Node.js； //官网下载
记事本：notepad＋＋ //一个文本编辑器

安装Hexo：

在cmd运行
$ npm install -g hexo-cli //安装完整hexo，
[$ npm install hexo //仅局部安装hexo，适用于熟悉npm的进阶用户]
$ git –version //检查安装及版本，hexo、git、npm同适用
