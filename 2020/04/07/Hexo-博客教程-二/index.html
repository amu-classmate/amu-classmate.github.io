Hexo 博客教程（二）

Hexo 博客教程（二）

基于Windows系统，本系列博文主要以实操为主，理论及部分非必要功能请查看Hexo官网文档

学习途径（视频）：B站

概要：接下来我们将进入实操，分为三部分，本篇博文为第一部分：建站

建站

１.在选定文件夹中新建所需要的文件：

在文件夹中按住Shift键，右键鼠标，点击“在此处打开命令控制窗口” （或也可以直接在地址栏输入“cmd”回车即可）

输入命令：
$ hexo init //新建一个网站，folder为博客本地文件名称

$ cd <folder>                //进入文件夹

$ npm install                //安装网站所需的依赖，会在文件夹生成一个package-lock.Json文件

Hexo新建完，会得到一个文件夹，里面包含文件

_config.yml            //配置文件

Package.json        //应用程序的信息

Scaffolds            //模板文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。

Source                //资源文件夹是存放用户资源的地方。

    drafts                //

    posts                //博客文章存放的文件夹

Themes                //主题文件夹。Hexo 会根据主题来生成静态页面。更换博客主题即为修改该参数

配置

１.打开notepad＋＋，将选定文件夹作为工作区

‘‘

2.更改网站数据

Title——标题    

Subtitle——小标题

Description——描述

Author——著作者

3.更改域名（URL）： https://博客名称.github.io //注意，该地址必须要与GitHub账号同名，与博客地址仓库同名，不然部署容易出错
